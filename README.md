# CMF-Paper
Code by Kieron Brandt for Exoplanet Thesis Project

- `survey.ipynb` contains code for reading and calculating CMFs from downloaded CSV files
- `benchmark.ipynb` contains code to show performance of Fast Abundance to CMF and benchmarking times
- `cmf_utility.py` contains utility functions for calculating CMFs
- `exosystem.py` contains classes to calculate CMF, mass, and radius
